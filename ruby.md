# Ruby lang

## hello world

`puts 'Hello ,world!'`

## commment

Use `#` to convert a line to comment.

## combine strings

`'hoge'+'fuga'` equals `'hogefuga'`.

## variables naming

Use `_` for combination like `user_name`.

## template literal

Use `#{}` to use variables in string.  
`"こんにちは、#{user_name}さん"`  
You must use `""` for template literal.  
`''` doesn't work.
