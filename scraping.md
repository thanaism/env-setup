# Web スクレイピング関連

## mecab

フリーの形態要素解析として mecab を使える。
`mecab-ipadic`がベースとなる辞書なので併せてインストールすることを忘れないようにする。

```bash
$ brew install mecab mecab-ipadic
$ echo 'すもももももももものうち' | mecab
すもも 名詞,一般,*,*,*,*,すもも,スモモ,スモモ
も 助詞,係助詞,*,*,*,*,も,モ,モ
もも 名詞,一般,*,*,*,*,もも,モモ,モモ
も 助詞,係助詞,*,*,*,*,も,モ,モ
もも 名詞,一般,*,*,*,*,もも,モモ,モモ
の 助詞,連体化,*,*,*,*,の,ノ,ノ
うち 名詞,非自立,副詞可能,*,*,*,うち,ウチ,ウチ
EOS
```
